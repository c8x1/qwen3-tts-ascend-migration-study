Speaker encoder：mel 到 embedding

状态：已核验 Qwen3TTSSpeakerEncoder 是 ECAPA 风格说话人编码器：输入侧契约是 24kHz 音频对应的 128-bin mel，forward 产出 speaker embedding。这里描述类与张量路径，不声称音频已在本机执行。

  • 证据 TGT-MODEL-001

Talker：text+codec 到 codec-0

状态：已核验 Qwen3TTSTalkerForConditionalGeneration 把文本与已有 codec 条件送入 decoder，并由 LM head 预测 codec-0 logits。codec-0 是 Talker 的主序列输出，不能与后续残差码本组混为一谈。

  • 证据 TGT-MODEL-002

MTP：15 个残差码本组

状态：已核验 code predictor 消费 Talker hidden state 与先前码本条件，补齐 15 个残差码本组；forward_sub_talker_finetune 返回残差码本损失。因而 codec-0 与 15 个残差码本组是两个相邻但不同的职责。

  • 证据 TGT-MODEL-004
  • 证据 TGT-MODEL-002

目标源码的 FP32 精度岛

状态：已核验 目标模型源码中，Talker RoPE 在关闭 autocast 后用 FP32 计算再恢复输入 dtype；RMSNorm 的方差用 FP32；eager attention 的 softmax 用 FP32 再转回 query dtype。这些是官方目标实现的局部精度范围，不是迁移映射。

  • 证据 TGT-MODEL-005
  • 证据 TGT-MODEL-006
  • 证据 TGT-MODEL-007
