目标项目 CLI 默认值

状态：已核验 官方目标 CLI 的默认设备、dtype 与注意力实现分别是 cuda:0、BF16 与 FlashAttention-2（FA2），解析后的值会传入 from_pretrained。这些是目标项目面向 NVIDIA/CUDA 的默认值描述，不是设备迁移建议，也不是任何替代后端的映射结论。
