SFT 训练循环

按固定 sft_12hz.py 的真实顺序讲解 BF16+FA2 默认、Accelerate ownership、embedding 汇总、Talker/MTP 损失与 AdamW 更新。

学习目标

  • 按源码顺序定位模型、配置、数据、优化器与 accelerator.prepare
  • 区分 prepared model 的已核验调用点与 wrapper/multi-process 待执行行为
  • 解释主 Talker 与 residual code predictor 的组合损失和更新顺序

前置知识

  • 熟悉 PyTorch 单卡张量与训练循环
已核验项目声明静态推断待真机验证
